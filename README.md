# 📍 Taggin

**Taggin** is a hyperlocal social platform connecting students within nearby campuses — 
built for real communities, not the whole internet.

🌐 Live: [taggin.app](https://taggin.app)

---

## 🚀 What is Taggin?

Taggin lets students discover and connect with people around them — 
on their campus, in their hostel, or nearby colleges.
No algorithms. No strangers from across the world. Just your people, nearby.

---

## ✨ Features

- 📍 Location-based user discovery
- 👥 Friend system — send, accept, remove
- 💬 Real-time chat
- 🏫 Campus-aware connections
- ⚡ Smooth, responsive UI

---

## 🛠️ Built With

- **Flutter** — Cross-platform mobile UI
- **Dart** — Programming language
- **Firebase Auth** — User authentication
- **Firestore** — Real-time database & chat
- **Firebase Hosting** — Deployment

---

## 📱 Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio or VS Code
- Android/iOS emulator or physical device

### Installation

```bash
git clone https://github.com/Ayushman00024/Taggin.git
cd Taggin
flutter pub get
flutter run
```
